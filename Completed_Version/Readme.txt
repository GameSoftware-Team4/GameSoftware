게임소프트웨어 4조 게임
탕탕 서바이벌 Readme

사용 unity version : Unity 2021.3.10f1

포톤 코드가 없어서 실행이 안될 경우 해결법
1. Photon폴더를 들어간다.
2. Photon Unity Networking을 들어간다.
3. Resources를 들어간다.
4. Photon Server Setting을 클릭한다.
5.Server/Cloud Settingsf를 클릭한다.
6.App Id Realtime에 아래의 코드를 입력한다. 그후 실행하면 된다.
코드 : 6fe03047-e000-40cc-801b-3fb3fa08ff30

게임 시작하는 법 
1. 게임을 실행한다.
2. Loading이 끝나면 선택을 한다.
Find Room ; 방을 찾는다
Create Room : 방을 만든다
Quit Game : 게임을 끈다

3-1. Create Room을 선택한 경우 
1. 방이름을 입력하고 만들 수 있다.
2. 바로 시작할 수도 있고 기다렸다가 다른 사람들이 들어오면 시작할 수 있다.
Start Room : 게임 시작
Leave Room : 방을 나간다

3-2. Find Room인 경우
1. 이미 생성된 방 중 선택한 뒤 들어간다.
2. 방장이 시작할 때까지 대기한다.

4. 게임을 즐기면 된다.

클리어 방법
제한시간 12분 동안 생존하면 된다.(3번까지 살아날 수 있다)

실패 조건 
시간을 못 버티고 죽으면 GameOver

아이콘 설명
하트모양 : 플레이어의 체력을 나타낸다.
탄알모양 : 총의 남은 탄환 수를 나타낸다.
시계모양 : 남은 시간을 나타낸다.

스테이지 업
일정시간이 지나면 스테이지 레벨이 상승한다.
이때 Stage Up이라는 UI가 나타난다.
변동사항
1. 스킬 레벨업 가능(하나만)
2. 플레이어 체력 회복 및 최대체력 증가
3. 몬스터의 스폰되는 수가 증가한다.

플레이어 데미지
몬스터와 접촉했을 경우 데미지를 받고 1초간 무적상태가 된다.

조작법
1. 이동
W - 전진
S - 후진
A - 왼쪽 이동
D - 오른쪽 이동
Q - 재장전
Shift 누르면서 이동키 - 대쉬
Spacebar - 점프
Tab - 몇번 죽었는 지를 알 수 있다.


마우스로 적을 조준할 수있므녀 왼쪽 클릭으로 쏠 수 있다.

스킬 사용법 ( 모든 숫자키는 위에 있는 숫자키만을 사용합니다.)
조건1 : 스테이지 업 할 때마다 하나씩 스킬 레벨을 올릴 수 있다.
처음에는 레벨이 0으로 사용할 수 없다.

레벨업 키 : 왼쪽 Alt를 누르면서 해당 스킬 키코드를 누르면 된다.

책 : 1번(최대 레벨업 3번까지 가능) 
설명 : 플레이어 주변을 돌면서 적을 공격한다. 레벨업 할 때마다 한개 씩 늘어난다. 

폭발 : 2번
2번을 누르면 발동한다. 폭발을 일어켜서 주변 적들에게 피해를 입힌다.
쿨타임 10초 

아이스 : 3번
3번을 누르면 발동한다. 어름 파동을 주변에 일으켜서 피해를 입히고 일정시간 동안 속도를 둔화시킨다.
쿨타임 10초

수류탄 : 4번 (한번 더 레벨업 가능)
4번을 누르면 수류탄을 굴린다. 굴러가다가 일정 시간이 지나면 폭발을 하고 주변 적에게 피해를 입힌다.
쿨타임 5초

수류탄 ver2
수류탄이 진화한 스킬로 기존과 동일하지만 10초간 적에게 데미지를 주는 화염을 주변에 생성한다.

궁극기 : R
R을 누르면 스킬이 시작되고 적에게 데미지를 주기까지 시간이 소요된다. 대신 그만큼 데미지가 강력한다.
넓은 범위의 약한 몬스터는 한방에 죽일 만큼 강하기 때문에 쿨타임이 다른 스킬들 보다 길다.
쿨타임 30초

터렛 : V
V를 누르면 스킬이 발동되며 플레이어 주변에 적을 공격하는 터렛을 하나 설치한다. 이 터렛은 12초 동안 유지된다.
쿨타임 25초

로켓
다른 스킬들과 달리 몬스터를 죽이고 떨어트린 경험치 볼을 먹어야 사용가능하다.
최대 3개까지 증가한다.
