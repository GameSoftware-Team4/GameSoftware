Player_jhj = 플레이어 용 데미지 관리용 스크립트

OnDamage(1f)형식으로 사용하면 플레이어에게 데미지가 들어감
데미지가 남은 체력보다 높을 경우 사망 함수 시작
사망시 해당 오브젝트 파괴
(기존 몬스터에 들어가 있는 스크립트와 거의 흡사함)
플레이어 클래스의 오브젝트를 몬스터 스크립트에서 선언 후 Player_jhj를 GetComponent한 후 OnDamage(float damage)를 실행
하면 플레이어에 데미지가 들어감.(예상)
 

Mob_jhj = 기존 LivingEntity에 해당하는 스크립트. 완전 몬스터 용으로 변경
LowMob_jhj = 저급 몬스터 용 스크립트. 체력과 공격력을 여기서 조정 가능

Skill_set
플레이어가 사용할 수 있는 스킬들을 모아둔 공간
스킬 총 6개에 대한 트리거를 만들고 다 false로 설정
레벨업을 했을 때 혹은 특정 조건이 됬을 때 해당 스크리트에서 UnlockSkill(int number)를 호출하는 것으로
true로 변경되서 스킬 사용 가능하도록 설정
스킬 가능 조건은 트리거가 true일 때와 해당 스킬을 저장한 키를 누르는 두 조건이 동시에 만족할 때만 작동

현재 구현된 스킬 : 북, 폭발
예정 스킬 : 반갈죽(타노스), 아이스(광역딜 속도둔화, 색변경), 화염병(투척후 터진 에리어의 일정 공간내 도트딜)

쿨타임 함수(미완성)

스크립트 이름 변경 
BookMove_jhj = 책 움직임 구현
BookSkill_jhj = 책 데미지 구현
